# 冷蔵庫24時 北極星

## 目指す体験

面倒になりがちな食材管理を「毎日ちょっと見たくなる娯楽」に変える。

ユーザーが冷蔵庫を開く前に、期限が迫る食材と次に取るべき行動を、短く・楽しく・明確に理解できることを最優先とする。

## プロダクト原則

1. **まず役に立つ**：演出より先に、期限・数量・行動が正しく伝わる。
2. **大げさで楽しい**：特定番組の台詞や固有演出を複製せず、オリジナルの緊迫ドキュメンタリー調を育てる。
3. **登録を軽くする**：将来はバーコード、レシート、写真認識で入力負担を減らす。
4. **安全を断定しない**：期限だけで可食性を保証せず、現物確認を促す。
5. **個人データを守る**：外部送信は明示的同意と必要性がある場合に限定する。

## 成功指標（正式公開後）

- 初回登録完了率
- 7日後の継続利用率
- 期限前に消費済みとなった食材の割合
- 1件登録にかかる時間

## 当面の優先順位

1. MVPの実機テストと登録導線改善
2. PWA化と期限通知
3. 家族共有・クラウド同期
4. 写真／バーコード登録
5. ナレーション音声・テーマのプレミアム化
