## 注意点

- `localStorage` のキー名を変える場合はデータ移行が必要。
- 期限日はローカル日付の正午として比較し、タイムゾーン境界のずれを避けている。
- `overrides.postcss` は監査対策。削除時は必ず `npm audit --omit=dev` を再実行する。
- 食品安全を保証する表現へ変更しない。
- 台本追加時は特定番組の固有台詞をコピーせず、オリジナル表現にする。
- 台本は `src/content/narration.ts` の該当配列へ追加し、在庫・期限ロジックへ混在させない。
- GitHub公開範囲や外部サービス連携は、ユーザー確認なしに変更しない。
- ブラウザ音声には男女メタデータがないため、音声名による低音候補判定は補助扱い。実機で本人が選べる導線を維持する。
- タイプ演出の速度は `src/components/typewriter-text.tsx` の `CHARACTER_INTERVAL_MS` で管理する。
- タイプ演出を文字ごとのCSSアニメーションへ戻さない。長文で描画欠落が再発する可能性がある。
